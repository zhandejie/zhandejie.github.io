使用hexo搭建个人博客

发表于 2016-08-15   |   分类于 学习指南   |     |   阅读次数

为了可以让自己在学习的过程中，能够有效的且稳定的进步，决定搭建自己的个人博客，用来记录自己的学习历程。除了学习以外，也会在博客中记录生活中的点滴。
回到正题，使用hexo搭建个人博客，推荐参考Hexo的官方文档
下面是我自己搭建的过程：

搭建需求

  • windows操作系统（本人由于工作环境限制,此条件非必须）
  • Node.js（hexo的环境要求）
  • Git（版本控制工具）

安装Node.js

  • Node.js在Node.js中文网下载

安装Git

  • Git下载地址: https://git-scm.com/download/win

配置环境变量

配置nodejs和git的环境变量
目的是可以全局使用npm/git命令

安装Hexo

上述步骤完成后，即可使用 npm 安装 Hexo
在命令行窗口输入 npm install -g hexo-cli命令

初始化

在你的博客根目录执行 hexo init 比如我的博客根目录是e:blog\

测试

到此Hexo的安装就完成了，可以执行hexo version，有输出证明安装成功

工作流

  • hexo new "title" 新建一篇博客
  • hexo generate 或者 hexo g 生成博客的静态页面
  • hexo server 或者 hexo s 拉起本地测试服务器
  • 访问 http://localhost:4000/ 查看生成的博客
